🛠️ Detalles Técnicos: Lógica de Ingredientes y Persistencia

Este contrato evoluciona la arquitectura de la serie al añadir:

Identificador de Masa (string): Uso de la variable tipoMasa para distinguir entre preparaciones de maíz (tradicionales), arroz (típicas de Olocuilta) o yuca, elementos fundamentales en la identidad salvadoreña.

Sistema de Reputación Inmutable: Funciones de darLike y darDislike para medir la aceptación popular de las recetas registradas en la red Base.

Optimización de Almacenamiento: Mantenemos un límite de 200 caracteres por registro, asegurando que el despliegue y las interacciones sean eficientes en términos de gas.
